# W-Notes

w-notes adalah sebuah web yang dibuat dari react untuk mencatat notes, notes yang ditampung disini tidaklah permanent karena tidak menggunakan database atau juga localstorage. 

## Fitur
- Create Notes
- Delete Notes
- Archiv Notes
- Search Notes

## Teknologi 
- ReactJS
- Tailwindcss
